Integrated Excel GUI model
==========================

This example is similar as :ref:`Integrated mAB GUI model` except
that the process parameters are sourced from `Excel` spreadsheets.

Check :ref:`Running example with bokeh serve` for instruction on how
to run examples with local `bokeh` server in order to build an
interactive GUI.
