bio_rtd.peak_fitting¶

Helper module for obtaining peak shape parameters from data points.

calc_emg_parameters_from_peak_shape¶

bio_rtd.peak_fitting.calc_emg_parameters_from_peak_shape(t_peak_start, t_peak_max, t_peak_end, relative_threshold)[source]¶

Calculate EMG parameters from characteristic data points.

Parameters

  • t_peak_start (float) – Peak start position.

  • t_peak_max (float) – Peak max position.

  • t_peak_end (float) – Peak end position.

  • relative_threshold (float) – Relative signal (compared to max) at start and end positions.

Return type

Tuple[float, float, float]

Returns

  • rt_mean – Mean residence time (== first momentum).

  • sigma – Standard deviation of Gaussian part.

  • skew – The rate of exponential part.
